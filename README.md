# claudemods arch freeze
tool to turn any arch distro immutable
currently not tested only coded but alot of work will be done to this soon

# Lock system (default state)
immutarch-lock

# Unlock for maintenance
immutarch-unlock

# Safe system updates
immutarch-update

# Manage snapshots
immutarch-snapshot create my-snapshot
immutarch-snapshot list
immutarch-snapshot restore my-snapshot

# Emergency recovery
immutarch-recovery
